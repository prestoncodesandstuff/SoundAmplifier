## STEP 12: CONFIGURE SIGNING

**In Xcode left panel:**

1. 👆 Click **SoundAmplifier** (project icon, blue, at very top)
2. Center panel shows settings
3. Look for tab: **"Signing & Capabilities"**
4. 👆 Click that tab
5. You see:
```
┌────────────────────────────────────┐
│  SIGNING                           │
│                                    │
│  ☐ Automatically manage signing    │
│     ↑ CHECK THIS BOX               │
│                                    │
│  Team: [Apple ID dropdown ▼]       │
│                                    │
│  Bundle Identifier:                │
│  com.example.soundamplifier        │
│     ↑ CHANGE THIS                  │
└────────────────────────────────────┘
```

6. 👆 Click the **☐ checkbox** (square box)
   - Checkbox becomes: ☑️ (checked)

7. 👆 Click **Team dropdown**
8. Select your Apple ID from list

9. 👆 Click in **Bundle Identifier field**
10. Delete current text: `com.example.soundamplifier`
11. Type new: `com.yourname.soundamplifier`
    - Example: `com.john.soundamplifier`
